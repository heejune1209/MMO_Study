# 스탯

피격 판정이 되었을 때 상대방의 체력을 깍는 로직을 어떻게 하면 좋을까?

Arrow에서 피격되는 대상의 HP를 가지고 와서 바로 깍아주는 방법도 있을 것이고

그게 아니면 피격되는 대상의 메서드를 호출해서 실제 체력이 깍이는 것은 해당 메서드에서 하는 방법도 있을 것이다.

일반적으로 피해자 쪽에서 메서드를 호출하는 방식이 조금 더 자연스럽다.

⇒ 일반적으로 나중에 가면 피해자 쪽에서 패시브 등이 있기 때문에 상대방의 대미지를 감소시키는 기능들이 있을 수도 있기 때문에 상대방 쪽 정보에 접근할 일이 많아져서 타겟의 함수를 호출하는 것이 조금 더 좋을 것이다.

일단 체력을 깎이 위해서는 스탯 정보가 필요하기 때문에 서버와 클라 공용으로 사용할 스탯 구조체를 protobuf에서 만들어보자

```protobuf
message StatInfo {
  int32 hp = 1;
  int32 maxHp = 2;
  float speed = 3;
}
```

향후 힘, 지능, 민첩 등도 위 구조체에 넣어주면 된다.

그리고 정보를 보낼 때 Stat의 정보도 보내는 상황이 발생할 것이기 때문에 objectInfo에도 추가해준다.
